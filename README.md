# Capital Economics - API Examples

(c) 2026 Capital Economics Ltd.

This repository contains examples of how to use the Capital Economics APIs.  There are two different APIs:

* [Data API](./data_api/README.md)
* [Content API](./content_api/README.md)

## Introduction

The **Data API** provides access to a range of economic data and forecasts and is available to clients with a **valid subscription** for use of this product.

The **Content API** provides access to the articles and publications in the Capital Economics platform and is available to clients with a **valid subscription** for use of this product.

For an overview of the APIs and Excel plugin see our [Data Capabilities page](https://www.capitaleconomics.com/data)
